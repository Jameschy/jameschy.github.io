hexo常用命令列表

插件安装

  1. npm install hexo -g #全局安装Hexo
  2. npm install hexo-bridge #安装hexo-bridge插件
  3. ssh-keygen -t rsa -C “***@foxmail.com” #生成GitHub key
  4. cd ~/.ssh #进入ssh目录
  5. cat id_rsa.pub #查看id_rsa密钥
  6. ssh -T git@github.com #查看id_rsa密钥配置检测
  7. git config –global user.name “jameschy”  #配置username
  8. git config –global user.email “***@foxmail.com”#配置email
  9. git config user.name #查看username配置
  10. git config user.email #查看email配置

Hexo执行

hexo d 发布到远端

常见错误：
$ hexo d
INFO Validating config
ERROR Deployer not found: git

修复命令：
npm install –save hexo-deployer-git

hexo init #hexo初始化

hexo clean #清除缓存

hexo new “博客标题” #创建“博客标题”的文章md文档

hexo new page “pageName” #新建页面

hexo generate #生成静态文件至public目录

hexo server #运行本地服务器预览（默认端口4000，’ctrl + c’关闭server）

hexo deploy #部署到远程站点github

hexo help # 查看帮助

hexo version #查看Hexo的版本；简写hexo v
