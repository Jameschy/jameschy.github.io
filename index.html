Easyadmin安装

应用环境：

1.phpStudy 8.1.1.3

2.WNMP：Windows10 + Nginx1.15.11 + MySql5.7 + PHP7.2.9nts 

框架版本：

1.ThinkPHP6.0

2.layui2.9.x

3.Echarts5.51

流程步骤：

1.开启phpStudy 必要环境后，创建本地网站目录

 2.将本地目录指向public后，填写好伪静态规则：

location / {
　　if (!-e $request_filename) {
　　　　rewrite ^(.*)$ /index.php?s=/$1 last;
　　}
}

3.选择对应网站，右键打开菜单，选择composer；如果弹出的命令窗口显示路径在public下，则输入命令cd.. ，退回到根目录，

4.在网站根目录，输入Composer命令，安装必要环境：.

composer install --ignore-platform-reqs

显示如图则表示安装成功：

5.创建本地数据库，

 

6.访问本地网址，进入安装界面，填写数据库信息如图：

注意，数据库地址修改为 localhost，选择覆盖数据库

7.安装完成后，进入登陆界面,输入上面设置的admin账号密码进行登陆：

 

 8.如果需要关闭验证码，参考：Easyadmin去除后台验证码登陆的方法介绍

function is_image_404_using_curl($image_url) {
    $ch = curl_init($image_url);
    curl_setopt($ch, CURLOPT_NOBODY, true);
    curl_setopt($ch, CURLOPT_FOLLOWLOCATION, true);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    curl_exec($ch);
    $status_code = curl_getinfo($ch, CURLINFO_HTTP_CODE);
    curl_close($ch);
    return $status_code == 404;
}

hexo常用命令列表

插件安装

  1. npm install hexo -g #全局安装Hexo
  2. npm install hexo-bridge #安装hexo-bridge插件
  3. ssh-keygen -t rsa -C “***@foxmail.com” #生成GitHub key
  4. cd ~/.ssh #进入ssh目录
  5. cat id_rsa.pub #查看id_rsa密钥
  6. ssh -T git@github.com #查看id_rsa密钥配置检测
  7. git config –global user.name “jameschy”  #配置username
  8. git config –global user.email “***@foxmail.com”#配置email
  9. git config user.name #查看username配置
  10. git config user.email #查看email配置

Hexo执行

hexo d 发布到远端

常见错误：
$ hexo d
INFO Validating config
ERROR Deployer not found: git

修复命令：
npm install –save hexo-deployer-git

hexo init #hexo初始化

hexo clean #清除缓存

hexo new “博客标题” #创建“博客标题”的文章md文档

hexo new page “pageName” #新建页面

hexo generate #生成静态文件至public目录

hexo server #运行本地服务器预览（默认端口4000，’ctrl + c’关闭server）

hexo deploy #部署到远程站点github；简写hexo d

hexo help # 查看帮助

hexo version #查看Hexo的版本；简写hexo v
